# Web-Mercator Viewer с постановкой меток

Приложение на **Godot 4.4.1** для просмотра карт в проекции Web-Mercator, добавления пользовательских меток и вычисления расстояния между двумя точками.

В папке APP находится исполняемый файл Web-Mercator.exe, предназначенный для запуска приложения без Godot Editor.

---

## Основной функционал

* Масштабирование и перемещение карты.
* Постановка и удаление пользовательских меток.
* Выбор одного из трёх внешних видов меток.
* Просмотр списка текущих меток.
* Быстрый переход к координатам.
* Автоматическое форматирование ввода координат.
* Измерение расстояния между двумя точками (метры и километры).
* Отображение масштабной линейки.
* Автономная работа: плитки карты загружаются при текущем зуме и сохраняются локально в папку tiles внутри пользовательской директории (user://tiles). После просмотра часть карты доступна без интернета.
  (C:\Users\user\AppData\Roaming\Godot\app_userdata\WEB Mercator)

---

## Интерфейс приложения

### **Верхний левый угол**

Отображаются:

* текущий зум карты;
* мировые координаты положения курсора;
* расстояние между двумя точками: в метрах и в километрах.

Для вычисления расстояния поставьте две точки **ПКМ** по очереди.

---

### **Нижний левый угол — быстрый поиск**

Содержит:

* два поля ввода координат (**N** и **E**);
* кнопку **Поиск**.

При вводе координат автоматически подставляется точка после двух цифр.
После нажатия «Поиск» карта перемещается к введённой координате с зумом **7**.

---

### **Правая часть интерфейса**

#### Список текущих меток

Отображает все добавленные точки.

#### Выбор внешнего вида меток

Три кнопки позволяют выбрать один из доступных стилей.

#### Переключатель «Ввод/удаление меток»

* Включён — можно добавлять и удалять метки.
* Выключен — управление картой (перемещение и зум).

#### Точная постановка меток

В нижнем правом углу:

* поле ввода **N**;
* поле ввода **E**;
* кнопка **Добавить метку**.

Позволяет поставить точку по точным координатам.

---

## Работа с метками

###  Добавление метки

1. Выберите стиль метки.
2. Включите «Ввод/удаление меток».
3. Наведите курсор на карту.
4. Нажмите **колёсико мыши**.

ИЛИ:
5. Введите координаты в нижние правые поля и нажмите **Добавить метку**.

---

### ➖ Удаление метки

1. Включите «Ввод/удаление меток».
2. Нажмите **ЛКМ** рядом с нужной меткой — она удалится.

---

## Управление картой

Работает, когда «Ввод/удаление меток» **выключен**.

### Перемещение

Зажмите **ЛКМ** и тяните — карта перемещается.

### Масштабирование

Колесо мыши:

* вверх — увеличить зум;
* вниз — уменьшить зум.

---

## Измерение расстояния между двумя точками

Работает независимо от режима «Ввод/удаление меток».

1. Нажмите **ПКМ** в первой точке.
2. Нажмите **ПКМ** во второй точке.

Расстояние отобразится в левом верхнем углу в метрах и километрах.

---

## Ввод координат

Все четыре поля ввода координат:

* автоматически вставляют точку после двух цифр;
* верхнее поле — широта (**N**);
* нижнее — долгота (**E**).

Пример ввода:

```
55 → 55. → 55.75
```
